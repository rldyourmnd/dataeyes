# dataeyes-docs

Документы по DataEyes: **коммерческое предложение (КП)**, материалы для заказчика, обоснования и сопроводительная документация (не код).

> Часть монорепо [`dataeyes`](../README.md). Технический сервис-бенчмарк — в [`../model-bench/`](../model-bench/).

## Назначение

Здесь хранятся бизнес/КП-документы: презентации, коммерческое предложение, спецификации и человекочитаемые отчёты по результатам бенчмарка DataEyes-моделей.

## Документы

- [`proposal/kp-dataeyes-partners-ru.md`](proposal/kp-dataeyes-partners-ru.md) — **Коммерческое предложение для партнёров** (RU): платформа DataEyes, каталог моделей и ориентировочные цены, результаты независимого бенчмарка (прогон `deep_eval`, 2026-06-10), модели сотрудничества, внедрение.

## Структура

- `proposal/` — коммерческое предложение и его версии
- `reports/` — человекочитаемые отчёты по прогонам бенчмарка (экспорт из `model-bench`)
- `assets/` — изображения, диаграммы, PDF (создать по мере наполнения)

Кладите сюда `.md` / `.pdf` / `.docx`. Технические артефакты прогонов остаются в `model-bench/` (S3/local), сюда попадают только оформленные документы.
